Design doc reviewed: pretender is a language-agnostic code quality CLI built on tree-sitter. Core insight: universal code model (Module/CodeUnit/Block/Branch/Node) with per-language .scm query files as adapters. Metrics are pure functions over the model. Three modes: guidance/tiered/gate. Roles (app/library/test/script/generated/vendor) are first-class — test code has stricter cyclomatic/nesting/cognitive but looser length/duplication. ABC scoring + smell weights from Flog. Structural duplication via normalised AST subtree hashing from Flay. Mutation testing wrapper from Heckle. SARIF output for GitHub Code Scanning inline PR annotations. Binary: pretender, config: pretender.toml. Rust + tree-sitter + clap + serde_sarif + git2 + rayon. MVP: Python/JS/TS/Rust + 4 metrics (cyclomatic, function-length, nesting, params) + init/check commands + pre-commit hook generator.
